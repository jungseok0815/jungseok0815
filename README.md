## 🚀 소개
환경 및 수자원 관리 시스템을 위한 엔터프라이즈급 애플리케이션을 전문으로 하는 백엔드 개발자입니다. 실제 세계에 영향을 미치는 견고하고 확장 가능한 솔루션을 구축하는 것을 즐깁니다.

## 💼 현재 작업 중인 프로젝트
- 🌊 **수자원 관리 시스템** - 베트남 하천 유역 관리를 위한 통합 솔루션 개발
- 🛰️ **위성 데이터 처리** - GEMS/GEO-KOMPSAT-2B 환경 데이터 작업
- 🗺️ **GIS 애플리케이션** - OpenLayers를 활용한 인터랙티브 매핑 솔루션 구축
- 👔 **Outfit Archive** - 소셜 미디어와 이커머스를 결합한 패션 테크 플랫폼 개발

## 🎯 주요 프로젝트

### 💧 수자원 관리 시스템
베트남 하천 유역 관리를 위한 엔터프라이즈급 시스템
- Spring Boot 기반 RESTful API 구축
- 유역 데이터 처리 및 시각화 (Chart.js)
- WEAP 소프트웨어와의 자동화 통합
- 실시간 데이터 모니터링 및 분석

### 🛰️ GEMS 위성 데이터 처리 시스템
환경 모니터링을 위한 위성 데이터 분석 플랫폼
- 오존 프로파일 데이터 처리
- JSON to NetCDF 형식 변환
- Panoply를 통한 과학적 시각화
- 대기질 데이터 실시간 처리

### 🗺️ GIS 기반 지역 선택 도구
한국, 중국, 인도, 동남아시아를 위한 인터랙티브 매핑 솔루션
- OpenLayers를 활용한 지도 시각화
- 좌표 기반 지역 선택 기능
- 공간 데이터 처리 및 분석
- 다중 레이어 관리 시스템

### 👔 Outfit Archive
스타일 공유와 이커머스를 결합한 패션 테크 플랫폼
- 소셜 미디어 기능 통합
- AI 기반 스타일 추천 시스템
- 사용자 맞춤형 패션 큐레이션
- 전자상거래 기능 구현

### 🤖 AI 매칭 시스템
